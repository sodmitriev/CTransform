Detailed Description

Buffer that carries data between two transformations.

This struct acts as a buffer that should be placed between two adjacent elements of transformation process. It implements a simple contiguous memory management. Buffer may be both written to and read from. Buffer keeps track of two memory cursors: read cursor and write cursor. During read-write cycle cursors are only moved forwards and never backwards. This may lead to memory space exhaustion. To free unused memory space, buffer_compact(buffer *) should be invoked.

◆ buffer_compact()

buffer_compact ( buffer *  this)

Remove processed portion of data from the buffer.

Parameters
thisA pointer to "this" buffer

Invoking this method allows additional data to be written into the buffer, by removing data that was already processed.

This method removes all data that is located between buffer beginning and read cursor (i.e. the data that is already read), and moves usable data to the beginning of the buffer, while adjusting read and write cursors.

Parameters
